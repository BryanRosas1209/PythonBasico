# COMPENDIO DE INVESTIGACIÓN: FUNDAMENTOS DE BASES DE DATOS RELACIONALES

'''
1. ¿Qué es una Base De Datos Relacional?
   Es un sistema que organiza la información en tablas que se conectan entre sí. 
   Su objetivo es estructurar los datos de forma lógica para evitar que se 
   repitan innecesariamente y facilitar su búsqueda.

2. ¿Qué es una Tabla en el contexto de Base De Datos Relacionales?
   Es la unidad donde se guarda la información. Se organiza en una cuadrícula 
   de filas y columnas; cada tabla se enfoca en un solo tema (como "Clientes" 
   o "Inventario").

3. ¿Qué es una relación en el contexto de Base De Datos Relacionales?
   Es el vínculo que une a dos tablas distintas. Nos permite conectar datos 
   sin mezclarlos todos en un solo lugar, como relacionar a un "Alumno" con 
   las "Calificaciones" que tiene en otra tabla.

4. ¿Qué representa una columna en el contexto de Base De Datos Relacionales?
   Representa una característica o atributo del dato. Por ejemplo, en una 
   tabla de "Autos", las columnas serian: Marca, Modelo, Color y Placa.

5. ¿Qué representa un renglón en el contexto de Base De Datos Relacionales?
   También llamado registro, representa una unidad completa de información. 
   Es una fila que contiene todos los datos específicos de un solo elemento 
   de la tabla.

6. ¿Qué es un motor de base de datos relacional?
   Es el software o programa encargado de administrar los datos. Es el 
   "cerebro" que procesa nuestras peticiones, guarda los archivos y 
   cuida que la información esté segura.

7. ¿Cuáles son los motores de bases de datos relacionales mas populares?
   Los más conocidos son MySQL, PostgreSQL, Oracle Database, 
   Microsoft SQL Server y MariaDB.

8. ¿Cuál es el motor de base datos ligero que no necesita un servidor?
   Es SQLite. Es muy popular porque guarda toda la base de datos en un 
   solo archivo común, lo que lo hace perfecto para aplicaciones móviles 
   y prácticas de programación.

9. ¿Qué significa SQL en el contexto de base datos relacionales?
   Significa "Structured Query Language" (Lenguaje de Consulta Estructurado). 
   Es el idioma universal que usamos para darle órdenes a la base de datos.

10. ¿Cuáles son los elementos de un sistema de información basado en BD?
    Se compone de cinco piezas clave: el Hardware (la compu), el Software 
    (el motor), los Datos (la info), los Procedimientos (reglas) y los 
    Usuarios (quienes usan el sistema).
'''
